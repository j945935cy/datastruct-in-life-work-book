# 專案名稱
datastruct-in-life-work-book

# 專案目標
請協助我建立一本可在瀏覽器閱讀的「資料結構電子網頁書」，主題是：

**《資料結構在日常生活與工作中的應用》**

這本書不是寫給資工本科生，而是寫給：
- 一般普羅大眾
- 對資訊觀念有興趣的人
- 老師、學生、上班族
- 想知道「資料結構到底有什麼用」的人

本書要用**白話、生活化、工作化**的方式，介紹資料結構的概念與應用，不要過度學術化，不要一開始就用太多公式或艱深術語。

# 書籍定位
這不是一本硬核教科書，而是一本「知識型、應用型、入門型」資料結構書。

請把資料結構解釋成：
- 幫助整理資訊的方法
- 幫助分類、查找、排序、安排、關聯的方法
- 幫助工作更有效率的方法
- 幫助理解軟體、系統、日常決策的方法

請避免一開始就寫成：
- 純定義堆砌
- 純程式碼導向
- 純演算法教科書風格
