請建立一個純 HTML、CSS、JavaScript 的靜態網站電子書專案，書名主題為：

《資料結構在日常生活與工作中的應用》

一般普羅大眾、學生、老師、上班族，不是資工本科生。

- 資料結構不是只給工程師
- 它和整理、分類、查找、安排、關聯有關
- 它其實天天出現在生活與工作中

# 技術限制

- CSS
- JavaScript

- chapters/ch01.html
