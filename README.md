# datastruct-in-life-work-book

以純 HTML、CSS、JavaScript 製作的靜態電子網頁書專案，主題是《資料結構在日常生活與工作中的應用》。本專案面向一般讀者、學生、老師與上班族，用生活化與工作化的方式介紹資料結構。

## 目前內容

- 第 1 到第 12 章章節頁
- 第 1 到第 12 章完整第一版內容
- 深淺色模式切換
- 字體大小切換
- 閱讀進度條
- 目錄頁最近閱讀高亮
- 回到頁面頂端按鈕

## 本機開啟

直接以瀏覽器開啟 index.html 即可閱讀。

## GitHub Pages

這個專案不需要建置流程，推送到 GitHub repository 後，可直接用 GitHub Pages 發佈根目錄內容。
